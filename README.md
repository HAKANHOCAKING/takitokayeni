# Takı Toka - Personalized Charm Necklaces

A modern e-commerce application for personalized charm necklaces built with Next.js 14, TypeScript, Tailwind CSS, and Zustand.

## Tech Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Lucide React** (Icons)
- **Zustand** (State Management)

## Project Structure

```
├── app/                    # Next.js App Router pages
│   ├── layout.tsx         # Root layout with fonts
│   ├── page.tsx           # Home page
│   └── globals.css        # Global styles
├── components/            # Component library
│   ├── atoms/            # Basic building blocks
│   ├── molecules/        # Composite components
│   └── organisms/        # Complex components
├── lib/                   # Utilities and stores
│   ├── store/            # Zustand stores
│   └── utils.ts          # Helper functions
└── types/                # TypeScript interfaces
```

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Design Philosophy

- **Mobile-first**: Responsive design starting from mobile devices
- **Minimalist**: Clean, uncluttered interface
- **Luxury-oriented**: Premium typography (Inter & Playfair Display) and elegant styling
